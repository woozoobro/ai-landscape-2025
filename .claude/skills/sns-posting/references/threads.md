# Threads 포스팅 가이드

## 플랫폼 특성

- **타겟**: 일반 대중, 캐주얼한 팔로워
- **톤**: 친근하고 가벼운
- **길이**: 500자 제한 (짧을수록 좋음)
- **핵심**: 빠른 훅 + 임팩트

## 포맷 규칙

### 길이
- 500자 제한이지만 **200-300자**가 이상적
- 스크롤 없이 한 눈에 보이는 게 베스트
- 길면 시선이 흩어짐

### 줄바꿈
```
짧은 문장.

다음 문장.

마무리.
```
- LinkedIn보다 더 과감하게 끊어도 됨
- 3-4줄 단위로 덩어리

### 첫 줄 (Hook)
피드에서 0.5초 안에 승부

**효과적인 Hook:**
- 질문: "~해보셨나요?"
- 숫자: "365일 동안..."
- 선언: "확실히 달라졌습니다."
- 공감: "솔직히 힘들었어요."

## 포스팅 유형

### 1. 한 줄 임팩트
```
[강렬한 한 문장]

[부연 1-2문장]

[링크 또는 마무리]
```

### 2. 미니 스토리
```
[상황 한 줄]

[전환점 한 줄]

[결론 한 줄]
```

### 3. 정보 공유 (초압축)
```
[이게 뭔지 한 줄]

핵심만:
- 포인트 1
- 포인트 2
- 포인트 3

[링크]
```

## LinkedIn → Threads 변환

LinkedIn 글을 Threads로 옮길 때:

1. **Hook만 살리기**: 첫 줄은 거의 그대로
2. **본문 압축**: 스토리 → 1-2문장으로 요약
3. **디테일 삭제**: 배경 설명, 부연 설명 과감히 제거
4. **CTA 단순화**: "자세한 건 링크에서"

### 예시

**LinkedIn (길게):**
```
Anthropic, OpenAI, Google.

올해 이 세 회사의 발표를 따라가다 보면
하루가 금방 지나갔습니다.

매번 새로운 모델이 나오고,
며칠 뒤엔 또 다른 회사가 치고 나오고.

그때마다 정리해서 팀원분들께 공유했는데요.
"와 바빠서 볼 시간 없었는데 정리해줘서 고마워요!"
그 말 한마디가 꽤 뿌듯하더라구요.
...
```

**Threads (짧게):**
```
2025년 AI 소식, 솔직히 다 따라가셨나요?

바빠서 못 보셨던 분들 위해
1년치 싹 정리해뒀습니다.

오늘 밤 8시, 인프런 라이브에서 만나요!

https://inf.run/r62T3
```

## 이모지 사용

- LinkedIn보다 자유롭게 써도 됨
- 단, 과하면 가벼워 보임
- 1-2개 포인트에만 사용

## 링크

- Threads는 링크 프리뷰가 잘 안 뜸
- 영상과 링크 동시 불가
- 영상 올릴 경우 → 첫 댓글에 링크

## 체크리스트

- [ ] 너무 formal하지 않은가?
