# LinkedIn 포스팅 가이드

## 플랫폼 특성

- **타겟**: 직장인, 전문가, 비즈니스 관계자
- **톤**: 전문적이되 인간적으로
- **길이**: 1,300자 내외 (3,000자 제한)
- **핵심**: 스토리텔링 + 인사이트

## 포맷 규칙

### 줄바꿈
```
한 문장.

한 문장.

짧은 단락은 2-3문장까지.
```
- 문장마다 줄바꿈 → 가독성 UP
- 단, 모든 문장에 줄바꿈은 오히려 산만

### 첫 줄 (Hook)
"더 보기" 클릭 전에 보이는 영역이 승부처

**좋은 Hook:**
- 숫자로 시작: "600명이 신청했습니다."
- 질문: "2025년 AI, 따라가셨나요?"
- 선언: "세 거인의 방향은 완전히 달랐습니다."
- 반전: "저는 실패할 줄 알았습니다."

**피할 Hook:**
- "This one thing changed my life..."
- "I have a confession to make..."
- "Stop scrolling! This is important!"

## 포스팅 유형

### 1. Story Post (경험담)
```
[상황/배경]
↓
[전환점/깨달음]
↓
[교훈/인사이트]
↓
[CTA 또는 마무리]
```

### 2. List Post (정보 공유)
```
[Hook: 왜 이게 중요한지]
↓
1. 첫 번째
2. 두 번째
3. 세 번째
↓
[마무리]
```

### 3. Observation Post (관찰/의견)
```
[현상 관찰]
↓
[나의 해석]
↓
[시사점]
```

### 4. Contrarian Post (반론)
```
[통념 제시]
↓
[반론/다른 시각]
↓
[근거]
↓
[결론]
```

## 마무리 (CTA)

**좋은 마무리:**
- 자연스러운 질문: "여러분은 어떠셨나요?"
- 깔끔한 선언: "그럼 이따 뵙겠습니다."
- 링크 제공 (필요시)

**피할 마무리:**
- "Agree? 👇"
- "Follow me for more!"
- "Like if you learned something!"

## 이모지 사용

- 0-2개가 적당
- 문장 끝에 감정 표현 정도: 🤣 😅
- 불릿 대용 가능: ✓ → •
- 🔥🚀💯 과다 사용 금지

## 해시태그

- 3-5개 적당
- 포스팅 맨 아래에 배치
- 너무 일반적인 것(#success) 보다 구체적으로

## 체크리스트

- [ ] 첫 줄이 "더 보기" 클릭하고 싶게 하는가?
- [ ] 하나의 메시지만 담고 있는가?
- [ ] 가짜 겸손/허세가 없는가?
- [ ] 줄바꿈이 적절한가?
- [ ] 마무리가 억지스럽지 않은가?
