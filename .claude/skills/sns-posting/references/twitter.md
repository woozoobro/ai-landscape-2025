# X (Twitter) 포스팅 가이드

## 플랫폼 특성

- **타겟**: 테크/비즈니스 커뮤니티, 정보 소비자
- **톤**: 간결하고 날카로운
- **길이**: 280자 (한글은 140자 체감)
- **핵심**: 임팩트 + 공유 가능성

## 포맷 규칙

### 단일 트윗
- 280자 제한
- 한글은 2바이트라 실제로는 140자 느낌
- **한 문장 = 한 트윗**이 이상적

### 스레드 (Thread)
긴 내용은 스레드로 쪼개기

```
1/ [Hook - 가장 강렬한 문장]

2/ [배경/맥락]

3/ [핵심 포인트 1]

4/ [핵심 포인트 2]

5/ [결론/CTA]
```

**스레드 규칙:**
- 1번 트윗이 제일 중요 (이것만 보는 사람 많음)
- 각 트윗이 독립적으로도 이해 가능하게
- 5-10개가 적당 (너무 길면 이탈)

## 포스팅 유형

### 1. 한 줄 인사이트
```
[날카로운 관찰 또는 의견]
```
- 리트윗/인용 유도
- 논쟁적이어도 OK

### 2. 정보 공유
```
[무엇에 대한 건지]

- 포인트 1
- 포인트 2
- 포인트 3

[링크]
```

### 3. 스레드 (딥다이브)
```
1/ 2025년 AI 삼국지, 정리해봤습니다.

세 거인의 방향은 완전히 달랐어요.

🧵 스레드로 풀어볼게요 ↓

2/ 먼저 Anthropic.

'일의 깊이'를 팠습니다.
Claude Code, MCP, Skills...
개발자 도구에 집중했어요.

3/ 다음 OpenAI.

'일상의 넓이'를 장악했습니다.
GPT-5, Sora, Atlas 브라우저...
우리 삶 곳곳에 스며들었죠.

4/ 마지막 Google.

'판의 크기'를 키웠습니다.
Gemini 3, Veo 3, Antigravity...
인프라 자체를 AI화했어요.

5/ 정리하면:

Anthropic = 일의 깊이
OpenAI = 일상의 넓이
Google = 판의 크기

같은 목표, 다른 전략.
2025년은 정말 정신없었네요.
```

## Hook 작성법

**강한 Hook:**
- 숫자: "365일, AI가 완전히 바뀌었다"
- 대비: "작년엔 비서, 올해는 동료"
- 질문: "따라가셨나요?"
- 선언: "확실히 말씀드립니다"

**피할 Hook:**
- "🧵 A thread on..."으로 시작 (본론부터)
- "I've been thinking about..." (너무 느림)
- "Hot take:" (식상함)

## 이모지 & 기호

- 🧵 스레드 표시
- ↓ 계속 읽으라는 신호
- → 화살표로 흐름 표시
- 불릿은 `-` 또는 `•`

과하게 쓰지 말 것

## 링크

- 링크 있으면 도달률 떨어짐 (알고리즘)
- 대안: 첫 댓글에 링크
- 또는 "링크는 프로필에" 유도

## 시간대

- 출퇴근 시간 (8-9시, 6-7시)
- 점심시간 (12-1시)
- 밤 (9-11시)
